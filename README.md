# HR Analytics Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive HR Analytics Dashboard built in Microsoft Excel. It provides insights into employee data through KPI Cards, Pivot Tables, Pivot Charts, and Slicers, helping users analyze workforce trends and key HR metrics.

---

## 📊 Dashboard Features

- Interactive Dashboard
- KPI Cards
- Pivot Tables
- Pivot Charts
- Slicers
- Dynamic Filtering
- Employee Demographics Analysis
- Salary Analysis
- Performance Analysis
- Hiring Trend Analysis
- Education Level Analysis
- Department Analysis
- Overtime Analysis
- Remote vs Office Analysis

---

## 📈 Key Performance Indicators (KPIs)

- Total Employees
- Average Salary
- Average Performance Rating
- Average Years at Company
- Total Departments
- Remote Employees

---

## 📂 Dataset

The dataset contains employee information, including:

- Employee ID
- Department
- Position
- Gender
- Age
- Salary
- Hire Date
- Years at Company
- Education
- Performance Rating
- Overtime
- Remote/Office
- City
